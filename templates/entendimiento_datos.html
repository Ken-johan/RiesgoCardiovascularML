Entendimiento de los Datos

Dataset Utilizado

El dataset utilizado para este proyecto proviene del repositorio UCI, específicamente adaptado para la población joven de Bogotá. Las variables incluidas son:

  • Edad (15-29 años)
  • Género
  • Nivel de actividad física
  • IMC (Índice de Masa Corporal)
  • Presión arterial
  • Consumo de tabaco/alcohol
  • Hábitos alimenticios
  • Nivel de estrés
